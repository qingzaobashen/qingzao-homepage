# 网页设计基础：从零开始学习网页布局

网页设计是前端开发的基础。本文从零开始，介绍网页布局的基本原理和常用技巧。

## 一、网页布局的基本概念

每个HTML元素都可以看作一个盒子（盒模型），由内到外包括：content、padding、border、margin。

## 二、常用布局技术

### 1. 浮动布局（Float）
使用 float 属性让元素向左或向右浮动。需要注意清除浮动。

### 2. 弹性盒子布局（Flexbox）
使用 display: flex 创建弹性容器，适合一维布局。

**常用属性**：
- flex-direction：排列方向
- justify-content：主轴对齐
- align-items：交叉轴对齐

### 3. 网格布局（Grid）
使用 display: grid 创建网格容器，适合二维布局。

## 三、响应式设计

响应式设计是指网页能自动适应不同尺寸的屏幕。

**实现方法**：
1. viewport meta标签
2. 媒体查询（Media Queries）
3. 流动布局
4. 弹性图片

## 四、常见页面结构

### 1. 三栏布局
头部、导航、主要内容、侧边栏、底部

### 2. 居中布局
水平垂直居中

### 3. 圣杯布局
经典的网页布局：头部、尾部、中间三栏

## 五、布局的常见问题

### 问题1：元素重叠
检查定位属性，清除浮动。

### 问题2：元素高度坍塌
清除浮动（clearfix）。

### 问题3：margin塌陷
使用padding或border分隔。

## 结语

掌握盒模型、Flexbox、Grid等核心技术，能帮你应对大多数布局需求。祝你早日掌握网页布局！
