# 卫生间防水与回填流程指南

卫生间是家里水患风险最高、返工成本最大的地方，尤其是下沉式卫生间。防水和回填做不好，会影响楼下邻居，损坏自家墙体，还会影响以后的使用体验。规范的施工流程是杜绝渗漏、沉降的关键。

## 一、回填方式与防水材料

### 1.1回填材料及优缺点
下沉式卫生间（沉箱）的管道在楼板下，通常需要进行回填，以找平地面并保护管道。

**1.建筑垃圾回填**
- 做法：直接使用装修产生的砖块、混凝 土碎块等废料进行填充。
- 优点：成本极低，取材方便。
- 缺点：材料沉重，增加楼板负荷，易导致沉降；碎块棱角尖锐，极易破坏防水层和管道，造成漏水；缝隙大易积存污水，滋生细菌和异味。
目前已被普遍认为是一种落后且不推荐的做法。

**2.炭渣回填**
- 做法：使用炭渣进行填充，炭渣是木料未充分燃烧的产物，具有结构紧密的特点。
- 优点：价格便宜，质量轻，不会给楼板带来过重的负担，颗粒小，不会伤害防水层和管道，吸水性好，密度均匀。
- 缺点：材质松软，要边填边压，还是有沉降的风险。

**3.陶粒回填**
- 做法：使用轻质、多孔的圆形陶粒进行填充，并在表层铺设钢筋网加固。
- 优点：材质轻，不会给楼板带来过重负担；吸水性好，有助于保持干燥，颗粒小且圆润，不会伤害防水层和管道。
- 缺点：材料成本和施工成本较高；施工工艺要求高，若未用钢筋网加固，陶粒易移位下沉，导致地面开裂。

**4.架空回填（预制板架空）**
- 做法：用砖砌筑支撑墩，然后盖上预制水泥板，形成一个架空层，再在水泥板上进行防水和铺砖。
- 优点：整体重量最轻；内部中空，便于日后检修管道；排水效果好。
- 缺点：施工相对复杂，成本高；隔音效果较差；架空层内若防水失效易积存水汽，可能滋生虫蚁。

**5.发泡水泥（回填宝）回填**
- 做法：使用专用发泡水泥材料现场浇筑，其流动性可自动填满所有缝隙。
- 优点：质地轻，流动性好，能包裹管道形成整体，无缝隙；具有一定的防水、隔音性能。
- 缺点：材料成本和人工成本最高；一旦固化后，维修极为困难，需整体凿开。

### 1.2 防水材料及优缺点
防水通常需做两道：回填前在沉箱底部做一次，回填找平后再在面层做一次。
1.柔性防水涂料（常见）
- 做法：在找平后的基层上涂刷聚氨酯、丙烯酸或聚合物水泥（K11）等柔性防水材料，一般涂刷2-3遍，墙角、管根等接缝处需重点处理。
- 优点：无缝连接，能适应基层的轻微开裂和变形；施工方便。
- 缺点：对基层平整度和清洁度要求高；涂刷厚度不易均匀，若太薄则易失效。

2.卷材防水
- 做法：铺设防水卷材（如SBS、丙纶布卷材），通过热熔或冷粘法与基层粘结。
- 优点：厚度均匀，防水性能稳定；施工速度快。
- 缺点：在管道、转角等复杂部位需要裁剪和拼接，处理不当易成为漏水点；对施工技术要求高。


## 二、标准施工步骤

**第一步：基层处理**

基层检查与修补：水电改造后，清理沉箱到坚实基面。检查墙地面交界、管根周围有没有空洞、裂缝，用堵漏王仔细刷所有阴角。

**第二步：第一遍防水涂刷**

首遍防水涂刷：整个沉箱底部地面做第一遍防水，墙面返高30-40厘米。这层防止结构性渗漏和砌体墙根部吸水。

**第三步：第一次闭水试验**

等防水层干透后，在沉箱里蓄水，水深不低于3厘米，持续24小时。一定要到楼下邻居家看对应天花板，确认没渗漏、水渍、湿痕。通过了才能进行下一步。

**第四步：红砖分格砌**

用红砖在沉箱里砌"井"字形方格。把回填区分成几个独立小单元，分散地面荷载，避免整体沉降。

**第五步：填充陶粒**

在红砖方格里填陶粒。陶粒多孔、质轻、吸水率低，减轻楼板负担，还能蓄水疏导，是理想的回填材料。别用建筑垃圾回填。

**第六步：布设钢筋网（增强整体性）**

陶粒层填平后，铺直径6mm左右的钢筋网，网格间距15-20厘米。钢筋层把上方水泥砂浆找平层拉成整体，防止开裂、下沉。

**第七步：水泥砂浆找平**

在钢筋网上浇筑水泥砂浆找平，做坚固基层。砂浆标号要合适，太高容易开裂，太低强度不够、会起沙。

**第八步：面部第二次防水（核心防水层）**

找平层干透后做第二次防水，这是关键一步。
- 墙面：淋浴区防水高度不低于1.8米，其他区域不低于30厘米，刷两遍。
- 地面：也刷两遍，和墙面防水形成整体。
- 细节：地漏、管根、阴阳角等薄弱地方重点多刷一遍。

**第九步：第二次闭水试验（最终验收）**

这是防水工程的"大考"。再蓄水24-48小时，到楼下复查。这次通过了，防水才算真的合格。

**第十步：制作防水保护层**

防水层验收合格后，马上用薄水泥砂浆做保护层。防止后续贴砖时，尖锐工具或沙石划伤防水层。

**第十一步：墙面拉毛处理（确保贴砖牢固）**

防水层表面太光滑，直接贴砖容易空鼓脱落。必须做拉毛处理（用水泥砂浆做粗糙表面），增加墙面和瓷砖粘结材料的抓合力。

## 三、关键监工要点与常见陷阱

| 关键环节 | 必须监工检查的事项 | 常见陷阱与后果 |
| :--- | :--- | :--- |
| **材料进场** | 核对防水涂料品牌、型号、保质期；检查陶粒干净无杂物。 | 用劣质防水涂料或不透气的刚性防水；用建筑垃圾代替陶粒。 |
| **闭水试验** | 自己或让物业到楼下查看，水位线做好标记。 | 施工方自己看并谎报结果，留下渗漏隐患。 |
| **回填工艺** | 确认做了红砖分格、铺了钢筋网。 | 直接倒陶粒，没分格没钢筋，沉降风险大。 |
| **细节处理** | 检查管根、墙角有没有用堵漏王加强，防水层刷得均匀吗。 | 细节处理马虎，成了以后渗漏的"突破口"。 |
| **墙面拉毛** | 完工后摸墙面，要有明显粗糙感。 | 图省事省略这步，后期瓷砖大面积空鼓脱落。 |

## 四、注意事项

1. 过程留证：让施工方对关键工序（特别是闭水试验）拍照或录像，发给你留存，作为监督和维权的依据。
2. 别省时省料：卫生间防水回填是"隐蔽工程中的隐蔽工程"，贪便宜或赶时间简化步骤，以后维修成本可能是当初省的几十倍，还会影响邻里关系。
3. 选专业团队：向施工方提出这些工艺标准，写进合同附件。专业团队会主动遵守这些规范。
