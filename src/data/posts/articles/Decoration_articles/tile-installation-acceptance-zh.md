# 瓷砖施工与验收

## 一、施工前准备：确认铺贴排版图

1. 确认铺贴排版图：
    - 这是施工的重要依据，根据图纸确认铺贴起始点、缝隙宽度、与门洞橱柜的对应关系，并优化小条砖、碎砖的隐藏位置（如家具下方、阴角处）。

---

## 二、施工过程监工要点

1. 清理瓷砖脱模剂：铺贴前检查瓷砖背面，如有白色粉状脱模剂，需用钢丝刷清理干净，确保粘结牢固。
2. 使用专业工具开孔：所有水管、插座开孔应使用专用开孔器，保证孔洞圆滑精准，避免暴力敲击导致的瓷砖开裂。
3. 及时清理与清缝：要求师傅在铺贴过程中随时擦净砖面水泥残留，并在填缝剂干透前进行清缝，为后期美缝打好基础。
4. 全屋通铺，慎用过门石：
    - 全屋瓷砖通铺可以使空间显大、连贯。除湿区外，尽量取消过门石。门口处应使用整砖或经过计算的大块砖切割，避免出现狭窄的“小条砖”。
5. 墙砖阳角做海棠角：
    - 所有墙砖转角（阳角）建议做“海棠角”（留2mm钝边），不要做脆弱的45°尖角。海棠角预留的缝隙经美缝后，美观且不易崩瓷。
6. 卫生间施工要点：
    - 坡度：使用激光水平仪辅助施工，确保地漏为全区域最低点，泼水测试排水迅速无积水。
    - 墙地对缝：墙面砖与地面砖的缝隙应尽可能对齐，提升视觉整体性。
    - 回字形地漏：地漏处瓷砖切割成“回”字形，加速排水。
    - 淋浴区下沉：淋浴区地面应比干区低2-3cm，实现自然挡水。
    - 单包套门洞：门洞内侧贴砖，只做外门套，更简洁美观。
7. 厨房施工要点：
    - 墙面规方找平：这是安装无挡水条台面的前提，需用靠尺仔细检查墙面平整度与垂直度。
    - 止逆阀处贴整砖：油烟机止逆阀应在一整块瓷砖上开孔安装，确保密封严密。
    - 巧用杂砖：被橱柜完全遮挡的区域可使用杂砖，节省成本。
8. 窗口预处理：
    - 贴砖前，用电锤将窗口内侧刨出深度和坡度，便于后期窗套安装更贴合。

## 三、完工后验收标准

验收黄金时机：在瓦工完成地面保护前！

| 验收项目 | 检验工具与方法 | 合格标准与问题处理 |
| :--- | :--- | :--- |
| 1. 空鼓率检查 | 空鼓锤（或金属勺）轻敲砖面四角和中心。 | 墙面砖：单片空鼓率≤5%，整体≤5%为合格。<br>地面砖：不允许出现空鼓，尤其是边角。<br>处理：发现边角空鼓可注浆补救，中心或大面积空鼓需更换。 |
| 2. 表面平整度 | 2米靠尺+塞尺。将靠尺贴于墙面/地面，用塞尺测量缝隙。 | 缝隙 ≤ 2mm 为合格（国标）。重点检查砖与砖的接缝处是否平整。 |
| 3. 垂直度与阴阳角 | 红外线水平仪、直角尺。 | 墙面垂直偏差 ≤ 3mm。阴阳角应呈90度，偏差 ≤ 3mm。 |
| 4. 接缝高低差 | 手指触摸、硬币边缘滑过接缝。 | 瓷砖接缝应平齐，无明显刮手感，用硬币可平顺滑动。 |
| 5. 坡度与排水 | 乒乓球、泼水测试。 | 将乒乓球置于卫生间地面，应自然滚向地漏。泼水后无积水。 |
| 6. 观感与破损 | 肉眼观察、强光手电侧照。 | 无崩瓷、开裂、明显色差、划痕。缝隙均匀一致，无水泥残留。 |
| 7. 功能孔位 | 目测。 | 地漏、水管接口处开孔圆滑，尺寸合适，安装后无过大缝隙。 |

> 特别提示：验收时随身携带硬币、乒乓球等小工具，进行快速测试。

---

## 四、验收后工作

1. 成品保护：验收合格后，督促施工方用专业加厚保护膜覆盖地面，并用胶带封边，防止后续施工划伤、污染。
2. 拍照留底：对全屋瓷砖铺贴效果，特别是水电管线走向的瓷砖区域进行拍照存档，方便日后安装柜体、洁具时避让管线。
