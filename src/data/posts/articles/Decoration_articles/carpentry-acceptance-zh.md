# 木工吊顶施工与验收

## 一、核心施工工艺

### 1. 龙骨框架

- 主龙骨间距控制在80-120cm之间，用膨胀螺丝与楼板固定牢。
- 所有螺丝在龙骨上弹线定位后安装，确保每颗螺丝都在龙骨上，受力均匀，避免石膏板虚挂。

### 2. 石膏板安装细节

- 拐角处用整板：所有吊顶转角必须用整张石膏板做“L”型切割安装，不能拼接。这里是应力集中点，整板能大幅降低开裂风险。
- 板缝错开与V型槽：相邻两层石膏板的接缝要错开，不能通缝。拼接处开V型槽，方便填嵌缝石膏，增加接缝强度。
- 底板托侧板：立面与平面交接处（如双眼皮吊顶），用“底板托住侧板”的工艺，避免垂直通缝。
- 双层/打底加固：有造型或需要高稳定性的区域，用欧松板打底再封石膏板，或直接用双层石膏板，强度会更高。

### 3. 特殊点处理

- 窗帘盒：用欧松板打底，再双面封石膏板。预留宽度根据窗帘类型定：电动双轨至少25cm，手动单轨至少15cm。
- 中央空调出风口：做加长出风口要提前定尺寸，两端各留8-10cm。出风口内部用欧松板加固，记得把风口内侧刷黑，视觉效果更统一。
- 衣柜上方：在柜体上方吊一个10cm左右的小平顶，方便后期装石膏线或做双眼皮收口，确保衣柜顶部严丝合缝。

### 4. 功能点位预留

- 筒灯/射灯开孔：水电交底时确定位置，施工时避开主龙骨。常规开孔尺寸7.5cm，间距建议90-120cm，距墙15cm以上。
- 检修口预留：空调内机、水管阀门、烟道止逆阀附近必须留标准检修口，位置要方便操作。
- 灯光与电器线路：所有需要预埋的变压器（如线性灯）、电动窗帘电机电源线，封板前预留到位并测试。

## 二、竣工验收标准：自检清单

| 类别 | 验收项目 | 合格标准 |
| :--- | :--- | :--- |
| **平整稳固** | 1. 整体平整度 | 用2米靠尺检查，缝隙 ≤ 3mm。 |
| | 2. 龙骨固定 | 摇晃没松动感，主龙骨间距合规。 |
| | 3. 石膏板固定 | 螺丝嵌入板面，没凸起、漏钉。 |
| **工艺细节** | 4. 转角是否为整板 | 所有转角无拼接缝。 |
| | 5. 板缝是否错开 | 两层石膏板接缝没重合。 |
| | 6. V型槽处理 | 拼接处已开V型槽。 |
| | 7. 螺丝防锈处理 | 所有螺丝都是防锈螺丝，或已点涂防锈漆。 |
| **尺寸与位置** | 8. 窗帘盒尺寸 | 宽度、深度符合设计，内侧平整。 |
| | 9. 空调出/回风口 | 尺寸、位置正确，内部加固完成。 |
| | 10. 灯具开孔位置 | 位置准确，避开龙骨，间距均匀。 |
| | 11. 检修口位置 | 位置合理，大小足够。 |
| **功能预留** | 12. 预埋线路 | 灯带变压器、电动窗帘电源等已预留并测试。 |
| | 13. 柜顶平吊 | 衣柜上方小平顶已完成，尺寸合适。 |
| **安全与规范** | 14. 材料核对 | 石膏板类型（如厨卫防水）使用正确。 |
| | 15. 整体观感 | 造型与设计图一致，弧线流畅，直角方正。 |
