# 油工验收指南

## 一、核心原则：分阶段验收

油工验收不能只在最后乳胶漆刷完再看。应该在两个重要节点验收：
1.  第一阶段：腻子打磨结束后，上乳胶漆之前 —— 这是检验墙面平整度的最佳时机。
2.  第二阶段：乳胶漆完全干透后，家具进场前 —— 这是检验最终饰面效果与功能的时机。

---

## 二、第一阶段验收：腻子打磨后（验收平整度）

此时墙面为纯白色腻子层，任何不平整都暴露无遗。一旦刷上乳胶漆，尤其是在有颜色的墙面上，不平问题将被掩盖，极难发现。

### 验收时间与环境：
- **时间**：在晚上进行。
- **环境**：关闭室内主灯，创造黑暗环境。

### 核心验收方法：侧光照射法
1.  **准备工具**：高亮度手电筒或碘钨灯。
2.  **操作方法**：将光源紧贴墙面，沿墙面平行照射。
3.  **合格标准**：在侧向强光下，墙面应呈现均匀的光影过渡，无明显的波浪、凹凸、砂纸痕。
4.  **问题表现**：如果出现如同水面涟漪般的明暗条纹，即为“波浪墙”，说明打磨不合格。

### 同步检查其他项目：
- **阴阳角**：用直角尺检查是否呈90度，是否笔直。
- **平整度**：用2米长靠尺贴墙，用塞尺检查缝隙。国标要求顺平误差≤4mm，但为达到更好效果（尤其安装定制柜处），应力争≤3mm。
- **细节**：检查开关插座周边、踢脚线位置、门套窗套边缘，腻子是否收得干净、平整。

若此阶段发现“波浪墙”或严重不平，必须要求工人重新打磨直至合格，否则不能进入刷漆工序！

---

## 三、第二阶段验收：乳胶漆完工后（最终效果检验）

乳胶漆完全干透后（通常刷完一周后），进行最终验收。

### 1. 检查平整度与光洁度（复检）
- **侧光复检**：即使腻子阶段合格，刷漆过程也可能产生流坠、刷痕。再次在白天自然光侧射条件下观察墙面。
- **触摸检查**：手背轻抚墙面，感受是否光滑，有无颗粒、毛刺。

### 2. 检查墙面强度（防掉粉）
- **摩擦测试**：用干净的深色抹布或袜子，在墙面稍用力来回摩擦几次。
- **合格标准**：抹布上几乎没有或仅有极其轻微的白灰。如果掉粉严重，说明腻子强度不足或乳胶漆附着力差，未来极易脏污且无法擦洗。

### 3. 检查观感与工艺细节
- **色差**：站在房间中央，观察墙面颜色是否均匀一致，有无透底、漏刷。
- **流坠与疙瘩**：查看墙面尤其是阴角处，有无油漆流下来的“眼泪”状凸起，或未化开的漆料疙瘩。
- **开裂**：仔细检查石膏板接缝处、新旧墙体交界处、线槽回填处，有无细微裂纹。
- **收边**：检查与天花、踢脚线、门窗套、柜体交界处，漆线是否笔直、清晰、无污染。

### 4. 检查功能区域
- **踢脚线位置**：确认墙面最底部的乳胶漆涂刷高度，是否低于计划安装的踢脚线高度，避免安装后露出未刷漆的墙面。
- **开关插座周边**：是否清洁无污染，边缘切割整齐。

## 四、常见问题、原因与处理

| 验收发现的问题 | 可能原因 | 处理建议 |
| :--- | :--- | :--- |
| 波浪墙 | 腻子批刮不平，打磨敷衍。 | 第一阶段发现，必须返工打磨。第二阶段发现，局部可打磨后修补，严重需重做。 |
| 严重掉粉 | 腻子质量差、未干透就刷漆、过度兑水。 | 需查明原因。如果仅是面漆问题可打磨后重刷；如果是腻子问题，可能需局部铲除重做。 |
| 墙面开裂 | 基层处理不到位（未挂网）、材料收缩不均。 | 标记裂缝，用嵌缝膏填补，贴接缝纸带，重新批腻子刷漆。 |
| 漆面流坠 | 刷子蘸漆过多，涂刷过厚。 | 打磨平整后，重新薄刷修补。 |
| 颜色不均/透底 | 兑水过多，覆盖力差；刷涂遍数不够。 | 重新涂刷足够遍数的面漆。 |

## 五、给业主的验收清单

**第一阶段（腻子后）验收清单：**
- [ ] 夜间侧光检查，墙面无波浪、光影平滑。
- [ ] 2米靠尺检测，平整度误差≤4mm（重点区域≤3mm）。
- [ ] 阴阳角笔直，呈90度。
- [ ] 所有细节边角（插座、墙角）处理平整干净。

**第二阶段（乳胶漆后）验收清单：**
- [ ] 自然光侧视，漆面颜色均匀，无流坠、无颗粒。
- [ ] 深色抹布摩擦测试，无明显掉粉。
- [ ] 全屋无可见裂纹。
- [ ] 踢脚线、门窗套等收边处漆线整齐，无污染。
- [ ] 墙面底部高度足够，能完全被踢脚线覆盖。
