# 风管机挑选与安装指南

## 挑选阶段：明确需求

![风管机挑选](/images/articles/central-ac-selection-install/3-风管机挑选.jpg)
### 1. 确定核心参数：匹数与能效
*   **制冷量（匹数）**：根据房间面积选，宁大勿小。参考下表，西晒、大面积玻璃窗、层高较高的话适当选大一点。
*   **能效等级**：首选APF新一级能效，虽然贵点，但长期用更省电。

| 匹数 (P) | 参考制冷量 (W) | 适用面积 (㎡) | 适用场景 |
| :--- | :--- | :--- | :--- |
| 1.5P | 约3500W | 10-18 | 小客厅、大卧室 |
| 2P | 约5000W | 18-28 | 常规客厅 |
| 3P | 约7200W | 28-35 | 大客厅、横厅 |
| 4P/大3P | 7500W以上 | 35-50+ | 超大客餐厅一体 |

### 2. 功能需求与品牌
*   **控制方式**：问清楚标配是基础按键线控器还是智能线控器（一般要加钱），智能的可以手机远程控制，方便很多。
*   **噪音值**：如果对声音敏感，问清楚内机运行噪音多少分贝。
*   **冷凝水提升泵**：安装位置离排水口远，或者吊顶薄做不了排水坡度（要1%），就得装提升泵，不然会漏水泡吊顶。顺便问有没有浮子开关（防漏水保护）。
*   **品牌选择**：根据预算和需求选：
    *   高端：大金、三菱电机，技术稳定，价格高。
    *   中高端/国产：格力、美的、海尔、日立，性价比高，市场保有量大。
    *   高性价比：海信、奥克斯、华凌等。

### 3. 费用谈判：问隐藏项
跟商家谈的时候，把这些费用写进报价单，避免后期加钱：
1.  **铜管费用**：标准安装包多少米？超出部分多少钱一米？
2.  **风口费用**：标配风口多长？做加长线性风口要加多少钱一米？
3.  **开孔开槽费**：打墙洞、线控器开槽埋线要不要加钱？
4.  **高空作业费**：高层装外机有没有这个费用？
5.  **电源与布线**：包不包含线控器到内机的信号线？空调电源线（建议单独一路4平方线）一般要水电工预留，提前协调好。

## 安装准备：材料验收与定位置

### 4. 到货与材料验收
*   **核对机型**：检查外包装和内外机铭牌上的型号是不是一致，和合同约定的对不对。
*   **检查外观**：看看机器有没有磕碰、变形。
*   **查验辅材**：这很重要！要求用品牌专用辅料包，或者确认这些材料：
    *   **冷媒铜管**：用脱磷无缝紫铜管，壁厚至少0.8mm，颜色亮。
    *   **保温棉**：厚度至少15mm，按压回弹快，阻燃等级B1级以上。
    *   **冷凝水管**：用PVC-U给水管，硬度高、不变形。
    *   **吊杆、管码**：用全金属（比如镀锌）的，结实防锈。

### 5. 室内机定位与防护
*   **安装位置**：内机不能紧贴房顶，要留回风跟检修空间，离顶部至少2-3cm。用双螺母+减震垫固定，减少噪音。
*   **防尘保护**：吊装好后，用随机带的防尘罩把内机出回风口包好，用胶带封严，别让装修灰尘进去。

## 安装过程：核心工艺监督

![风管机安装](/images/articles/central-ac-selection-install/4-风管机安装.jpg)
### 6. 管道施工
*   **横平竖直**：冷媒管和冷凝水管要铺得横平竖直，铜管转弯用专用弯管器，别用手直接折，容易折瘪。
*   **充氮焊接**：焊接铜管时要充氮气保护，防止内部氧化，不然焊渣会损害压缩机。
*   **排水坡度与通气孔**：冷凝水管要有大于1%的向下坡度，最高点留排气孔，保证排水顺畅没异响。装完做灌水试验。
*   **保温严密**：所有铜管、冷凝水管接口、阀门都要用保温棉包严，用专用胶带粘好，防止冷凝水。

### 7. 电气与系统调试
*   **抽真空**：系统连好后，用真空泵抽真空至少20分钟（看管路长度），把空气和水分抽干净，这步很重要。
*   **开机测试**：现场试制冷/制热，看线控器功能好不好，听内机运行有没有奇怪的噪音。

## 收尾验收与后续衔接

### 8. 安装收尾
*   **孔洞封堵**：所有穿墙孔用发泡剂填严，特别是外墙的，防止雨水、虫子进去。
*   **标注尺寸**：装完让师傅在防尘罩上标出出风口、回风口、检修口的尺寸和位置，方便木工吊顶开孔。
*   **部件保管**：测试没问题后，把线控器拆下来自己保管，等墙面装修完再装，别弄脏弄坏了。

### 9. 与装修工序的衔接
1.  **水电阶段**：让水电工预留空调专用电源线（4平方），根据商家给的深度，给线控器底盒开槽预埋。
2.  **吊顶阶段**：把风口尺寸告诉木工，确保开孔准。吊顶做完后，让商家来量尺寸定制风口面板。
3.  **最后安装**：墙面刷漆、保洁做完后，通知师傅来装风口面板和线控器，做最终调试。
