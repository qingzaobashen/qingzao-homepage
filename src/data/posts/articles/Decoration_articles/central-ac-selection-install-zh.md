# 中央空调挑选与安装指南

## 品牌与安装决策
![中央空调挑选](/images/articles/central-ac-selection-install/1-中央空调挑选.jpg)
![风管机挑选](/images/articles/central-ac-selection-install/3-风管机挑选.jpg)

1.  **品牌选择**
    *   不要只看价格！报价太低的，很可能是劣质材料，后期维修成本高到离谱。
    *   主流品牌参考：
        *   日系：大金、日立、三菱电机、东芝，技术稳定，节能静音。
        *   国产：格力、美的、海尔、海信，性价比高，服务网点多。
        *   美系：约克、开利、特灵、麦克维尔，大机组和水系统有优势。
    *   重点看本地售后服务网点多不多，口碑怎么样，可能比品牌本身更重要。
    *   确认是否为官方授权经销商，避免购买串货机、翻新机。

2.  **室外机类型：按安装条件选**
    *   顶出风：散热好，适合外机位宽裕的户型，需要上方无遮挡，适合楼顶、阳台或设备平台。
    *   侧出风：散热效率略低于顶出风，适合较小空间，家用最常见，对空间深度要求小，确保侧面和背面通风就行。
    *   让商家根据你家外机位的尺寸、通风和物业规定推荐。
    *   风扇类型：小户型低负荷时单风扇噪音小更节能；双风扇在高负荷时效率更高、散热能力更强。

3.  **技术参数**
    *   内外机超配比：超配比=外机的匹数÷内机总和的匹数，大部分中央空调超配比在120%左右，个别品质好的可做到130%左右。如果经常多台内机一起开，需注意这个数字。
    *   APF数值：APF值越高，中央空调能效等级越高，APF值≥5.0才是一级能效。
    *   变频类型：直流变频比交流变频省电20%左右，且噪音更小。
    *   压缩机类型：涡旋式压缩机寿命长、噪音小、运转稳定；双转子压缩机低频运转稳定，省电实惠。外机匹数大的涡轮式压缩机优势更明显，≤160的考虑双转子压缩机，≥160的可考虑涡旋式压缩机。

4.  **关键组件**
    *   风口材质：别用铝合金（冬天会滴水），用ABS材质，防结露、不变形、噪音小。验收时记得用手摸风口边缘，毛刺多的返工！
    *   保温棉：劣质保温棉让空调成为"电暖器"。选B1级阻燃橡塑材质，厚度在10-15mm为佳，按压回弹快、没味道。接缝处用专用胶粘合。回弹快的质量好，一捏就扁的赶紧退货。推荐品牌：福乐斯、华美。
    *   冷凝水提升泵：冷凝水管坡度 <1%，就得装提升泵，不然会漏水泡吊顶，维修要拆吊顶。水泵主要是给安装兜底，如果排水坡度做的好，水泵就没必要，但人工操作可能有误差，水管很长或安装师傅技术不行时，安装一个很有必要。避免使用外置或机械水泵。
    *   冷媒铜管和分歧管：要用全新的，铜管壁厚至少0.8mm，分歧管选原厂或优质配件。铜管推荐品牌：中佳、飞轮、海亮。
    *   电辅热：电辅热≈"电热丝吹风机"，非常耗电，带上电辅热的中央空调通常是压缩机不行，销售如果把"电辅热"当成卖点，快跑！
    *   排水管：PVC-U材质，推荐品牌：公元、中财。

## 合同签订与安装准备

合同写清楚，后期少扯皮。

1.  **签订合同：明细要全**
    *   写清增项：打孔费、高空作业费、超长度铜管费（通常包5-8米）、信号线加长费等提前谈妥，避免增项。
    *   产品明细：报价清单要详细，包括主机、内机价格，保温棉、铜管、分歧管、冷凝水管、风口材质的品牌型号写进合同。
    *   明确保修期：是厂家还是施工方保修。写明条款：包括整机和压缩机保修期。
    *   隐蔽工程质保：隐蔽工程质保时间非常重要，避免漏水拆木工吊顶、重新做油漆的大开销。
    *   设备质保：国产中央空调品牌普遍6-10年，合资品牌在3年左右。

2.  **付款方式**
    *   签订合同付定金时记得少付，一般预付款在总金额的20%-30%左右，等到室内机、室外机、遥控器以及其他零部件全部到场后再付清，千万别没看见东西就付款，唔好自己的钱包。

3.  **安装团队**
    *   所谓"三分产品，七分安装"，最好暖通公司有自己的专业安装团队。
    *   暖通公司成立时间：尽量选择在本地做了十多年的老牌暖通公司，专注一个品牌的。

4.  **其他注意事项**
    *   外机电源线：基本上中央空调内机电源线是由中央空调经销商承担的，外机电源线是由装修公司承担的。
    *   能耗数据：询问同开率高/低时，这款低频/高频的能耗数据，空调表现好不好，需要真实的使用数据来说话。
    *   特殊户型配置：开放式厨房/50㎡以上的大客厅，需要合理配置冷热方案，避免"小马拉大车"的情况。

## 安装与验收要点
![中央空调安装](/images/articles/central-ac-selection-install/2-中央空调安装.jpg)
![风管机安装](/images/articles/central-ac-selection-install/4-风管机安装.jpg)

1.  **内机安装：防尘和减震**
    *   防尘：内机要有防尘保护，用原厂保护膜或防尘罩包好，避免装修灰堵住滤网。
    *   减震：吊杆用双螺母固定，顶部加减震垫，与天花板间隔1cm，能减少室内共振噪音。

2.  **管道施工：看工艺**
    *   充氮焊接：焊接铜管时要充氮气，防止内部氧化，不然氧化掉皮会损坏压缩机。
    *   冷凝排水管测试：冷凝水管要有至少1%的坡度，排水口做U型存水弯防异味，装完做灌水试验。
    *   吊卡：间距0.8-1米，不然会导致铜管下垂，制冷剂泄露。用不锈钢或镀锌的，转弯处0.3米内要加固。选不锈钢材质，防锈耐用。

3.  **系统保压与抽真空**
    *   保压测试：管道装好后充40kg高压氮气（约4.0MPa），保压72小时，压力不降才算密封好。
    *   充冷媒前抽真空，时间要够（根据管道长度，至少30分钟），不然影响制冷和压缩机寿命。

4.  **竣工验收：逐项检查**
    *   对齐：检查风口安装平不平，和吊顶对齐没（错位 <1cm），边缘有没有毛刺。
    *   运行测试：每台内机出风正常吗？制冷/制热快不快？有没有异常噪音或振动？最低温度运行时，风口和铜管接口有没有滴水？
    *   要发票、保修卡、安装确认单，确认厂家已经注册，启动官方保修。
