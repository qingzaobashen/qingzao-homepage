# Carpentry Ceiling Construction and Acceptance

## 1. Core Construction Craft

### 1. Keel Frame
- Main keel spacing controlled between 80–120 cm, fixed firmly to the floor slab with expansion screws.
- All screws are installed after marking positions on the keel, ensuring each screw is on the keel, with even force, avoiding hollow-hung gypsum board.

### 2. Gypsum Board Installation Details
- Whole board at corners: All ceiling corners must use a whole gypsum board cut into an "L" shape, not spliced. This is a stress concentration point; a whole board greatly reduces cracking risk.
- Staggered joints and V-groove: The joints of adjacent gypsum board layers must be staggered, not aligned. Open a V-groove at the joint for filling joint compound, increasing joint strength.
- Bottom board supports side board: At the junction of vertical and horizontal surfaces (e.g. double-layer ceiling), use the "bottom board supports side board" craft to avoid vertical through-joints.
- Double-layer / backing reinforcement: For areas with shapes or requiring high stability, use OSB backing then cover with gypsum board, or directly use double-layer gypsum board for higher strength.

### 3. Special Point Treatment
- Curtain box: Use OSB backing, then double-sided gypsum board. Reserved width depends on curtain type: motorized double track at least 25 cm, manual single track at least 15 cm.
- Central AC outlet: For an extended outlet, determine size in advance, leave 8–10 cm at each end. Reinforce the inside of the outlet with OSB, and remember to paint the inside of the outlet black for a more unified visual effect.
- Above wardrobe: Hang a small flat ceiling about 10 cm above the wardrobe, convenient for later installing crown molding or double-layer ceiling finish, ensuring the top of the wardrobe fits tightly.

### 4. Functional Point Reservation
- Downlight/spotlight holes: Determine positions during plumbing/electrical handover, avoid main keels during construction. Conventional hole size 7.5 cm, spacing recommended 90–120 cm, at least 15 cm from wall.
- Access panel reservation: Standard access panels must be reserved near AC indoor units, water pipe valves, and range hood backdraft dampers, with convenient access.
- Lighting and appliance lines: All transformers to be embedded (e.g. linear lights), motorized curtain motor power lines must be reserved and tested before closing the boards.

## 2. Final Acceptance Standards: Self-check List

| Category | Acceptance item | Pass standard |
| :--- | :--- | :--- |
| **Flat & stable** | 1. Overall flatness | Check with 2-meter straightedge, gap ≤ 3mm. |
| | 2. Keel fixation | No loose feeling when shaking, main keel spacing compliant. |
| | 3. Gypsum board fixation | Screws embedded in board surface, no protrusion, no missing nails. |
| **Craft details** | 4. Whole board at corners | All corners have no splice joints. |
| | 5. Joints staggered | Two gypsum board layers have no overlapping joints. |
| | 6. V-groove treatment | V-groove opened at joints. |
| | 7. Screw rust prevention | All screws are rust-proof screws, or treated with rust-proof paint. |
| **Size & position** | 8. Curtain box size | Width, depth match design, inner side flat. |
| | 9. AC outlet/return | Size, position correct, internal reinforcement done. |
| | 10. Light hole position | Position accurate, avoids keels, even spacing. |
| | 11. Access panel position | Reasonable position, sufficient size. |
| **Function reservation** | 12. Embedded lines | Light strip transformers, motorized curtain power, etc. reserved and tested. |
| | 13. Wardrobe top flat ceiling | Small flat ceiling above wardrobe completed, proper size. |
| **Safety & standard** | 14. Material check | Gypsum board type used correctly (e.g. waterproof for kitchen/bath). |
| | 15. Overall appearance | Shape matches design drawing, smooth curves, square right angles. |
