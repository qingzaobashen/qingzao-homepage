# 地暖知识与线上筛选

## **01  地暖的组成**
![地暖系统结构](/images/articles/floor-heating-online-selection/1-地暖系统结构.jpg)

### **核心组件速览**
想象地暖是一个人体：
*   **锅炉（心脏）**：提供热源，烧热水。**选两用炉**，可同时满足采暖和家庭热水需求。
*   **管道（血管）**：热水在其中循环，将热量传递到地面。
*   **分集水器（中枢神经）**：控制热水分配到各个房间回路，并汇集回水。
*   **温控系统（反射神经）**：通过温控器感知室温，指挥系统运行，实现分室控温。
*   **保温系统（羽绒服）**：**最为关键却最易被忽视**，它决定了热量是向上温暖你家，还是向下浪费到楼下邻居。优秀的保温能直接降低后期30%以上的使用成本。

### **关键参数**
1.  **锅炉**：
    *   **类型**：普通炉、预混炉和冷凝炉。**冷凝炉**贵一点，但比常规炉更节能（热效率>100%），长期使用更省钱。（什么是冷凝炉？将燃烧后的烟气冷凝，吸收其中热量，而普通炉则是直接排掉了）
    *   **看编号辨真伪**：真正的全预混冷凝两用炉，编号通常是 **“L”开头，第5或6位是“Q”** ，如LL1GBQ18-B1JG。防止被“挂羊头卖狗肉”。
    *   **关键参数**：
        *   **功率**：参考公式：**采暖面积 ÷ 10 = 所需最小功率（KW）**。18KW：90平以下（采暖面积）；24KW：90-150平；28KW：150-220平；35KW：220-310平。
        *   **最小热输出功率**：越小越好，可以低功耗供暖，炉子不会频繁启停；好一点儿的能做到3KW。
        *   **生活热水量**：指每分钟能将多少升水升温25℃。越大越好，**建议选择≥12L**，才能保证洗澡时水量充沛、水温稳定。
    *   **品牌**：推荐德国威能、菲斯曼、日系林内、国产小松鼠、沃乐夫；
![冷凝炉](/images/articles/floor-heating-online-selection/2-冷凝炉.jpg)

2.  **管道**：
    *   **材质**：**PERT管**性价比高，满足家用，三/五层阻氧最好，减少细菌滋生；主管使用**PE-X系列**,追求极致可全用PE-X。
    *   **管径**：常规为16mm或20mm管。若采用两联供（地暖+空调）系统，建议使用20mm管以减小水流阻力。
    *   **关键要求**：一定要选 **“三/五层阻氧”** 管。它能防止氧气渗入管道滋生细菌、形成生物粘泥，避免地暖结垢堵塞，影响散热效果。
    *  **品牌**：国产：伟星、保利、日丰；合资：柯诺德、瓦普斯、乔治、瑞好；
![PERT地暖管](/images/articles/floor-heating-online-selection/3-pert地暖管.jpg)
![地暖盘管](/images/articles/floor-heating-online-selection/9-地暖盘管.jpg)

3.  **保温材料（重中之重！）**：
    *   **保温板**：推荐**白晶板（XPS挤塑板）**，环保无味，厚度至少2cm，优选3cm。一定要认清楚，不要被用成了普通的白板。蘑菇板不行，影响散热，且后期容易开裂。
    *   **反射膜**：必须是**纯铝镜面反射膜**，作用是：均衡热传导（真铝层的厚度决定效果），越厚越好，防止混凝土腐蚀保温板；劣质无纺布反射膜几个月后就会腐蚀失效。辅助安装和防水防潮注意：假冒的纯铝很容易混凝土的腐蚀。反射膜拼接的地方重叠5cm以上，并用胶带固定。
    *   **边界保温条**：沿墙铺设，防止热量向墙体流失。
![保温板](/images/articles/floor-heating-online-selection/4-保温板.jpg)
![边界保温条](/images/articles/floor-heating-online-selection/7-边界保温条.jpg)

4. **分集水器+温控**
![分集水器](/images/articles/floor-heating-online-selection/5-分集水器.jpg)
    *   **分集水器**：负责将热水分配到各个房间回路，并汇集回水。材质：优选一体锻造黄铜，壁厚足（≥4mm），阀芯选择高位或中位阀芯，不易堵塞。安装位置：优先考虑厨房水槽下、卫生间台盆下等靠近排水口的位置，便于日后检修。
    *   **温控系统**：实现分室独立控温。在每个主要房间安装温控面板，可根据需要设定不同温度，节能又舒适。

## **第二阶段：一张表进行线上调研与筛选**

### **第一步：调研前——明确你的需求**

1.  **明确核心需求**：
    *   **房屋情况**：地址、建筑面积、**采暖面积**（需扣除卫生间、阳台、落地柜下方等不铺管区域）。
    *   **功能需求**：采暖与生活热水两用；需要**分室独立控温**。
    *   **热水需求**：明确家中淋浴花洒、热水龙头数量（如2个花洒+3个龙头），这关系到锅炉生活热水能力的选取。
    *   **预算范围**：设定一个初步的心理价位区间，如90㎡，2w左右，锅炉价格占50-60%。

2.  **品牌矩阵**：
    *   **壁挂炉**：威能、菲斯曼、博世、阿里斯顿、林内、小松鼠等。
    *   **控制系统**：丹佛斯、嘉科米尼、霍尼韦尔、曼瑞德、西门子等。
    *   **管道**：瑞好、欧博诺、乔治费歇尔、伟星、保利、日丰等。
    *   **保温辅材**：绿羽、陶氏化学、欧文斯科宁等（**保温是节能关键，务必关注**）。

### **第二步：核心工具——《地暖系统配置调研表》**

将下面这份表格发给意向商家，要求他们逐一填写。

**地暖系统配置与施工细节调研表**

| 考察大项 | 具体调研项目 | 你的疑问与参考答案 |
| :--- | :--- | :--- |
| **1. 锅炉配置** | **类型与型号** | 请问是常规炉还是**冷凝炉**？请提供具体型号。（**提示**：正品全预混冷凝炉型号常以“L”开头，第5或6位是“Q”） |
| | **最大/最小功率** | 对于93㎡采暖面积，推荐多大功率？**最小输出功率**是多少？（越小，生活热水越稳，采暖越节能） |
| | **品牌** | 具体是哪个品牌哪个系列？ |
| | **一分钟热水量** | 升温25℃时，每分钟能产多少升热水？**建议选择≥12L**，以保证多点用水舒适。 |
| | **价格** | 锅炉设备部分单价是多少？ |
| **2. 分集水器** | **材质与工艺** | 是**一体锻造黄铜**吗？壁厚是否≥4mm？ |
| | **品牌** | 是什么品牌？（如丹佛斯、嘉科米尼、曼瑞德等） |
| | **阀芯** | 阀芯是高位、中位还是低位？（高位阀芯不易堵塞） |
| | **流量计** | 是否配有可视流量计？（便于调节平衡和诊断） |
| **3. 温控系统** | **温控器/执行器品牌** | 分室温控采用什么品牌？是否支持手机APP控制？ |
| **4. 管道系统** | **主管/盘管材质** | 主管是否为PEX管？盘管是否为**PERT五/三层阻氧管**？（阻氧管防结垢） |
| | **品牌与规格** | 分别是什么品牌？管径是多少？（如20mm或16mm） |
| **5. 保温结构** | **保温板** | 用什么品牌和材质的保温板？（**必须是白晶板/XPS挤塑板**，厚度≥2cm） |
| | **反射膜** | 是**纯铝镜面反射膜**吗？ |
| | **边界保温条等** | 是否包含**边界保温条**、**硅晶网**、**铝箔胶带**？ |
| **6. 施工工艺** | **盘管铺设** | 盘管间距多少？（标准15-20cm）地面部分**是否整管无接头**？ |
| | **伸缩缝** | 面积大于30㎡或长度超6米时，**是否设置伸缩缝**？ |
| | **回填材料** | 回填用什么材料？**必须为豆石混凝土**（水泥:沙:豆石≈1:2:3）。 |
| | **打压测试** | **是否做打压测试**？压力多大？保压多久？（标准：0.6-0.8MPa，保压至装修结束） |
| | **施工标准** | 施工参照什么规范？（如JGJ142-2012） |
| **7. 价格与售后** | **材料人工单价** | 除锅炉外，**每平米采暖面积的综合单价**是多少？（含所有管、辅、工） |
| | **总价构成** | 请列出：锅炉价 + 采暖面积单价 × 面积数 = 最终总价。 |
| | **保修政策** | 锅炉、管道、分集水器、温控器、安装工程**分别保修多久**？ |

### **第三步：分析回复并对比3家**

收到几家回复后，横向对比分析，重点看差异：

1.  **优先淘汰不专业者**：
    *   对表格内容避而不答、只催你线下谈的。
    *   对**保温材料**（白晶板、纯铝膜）和**施工细节**（伸缩缝、豆石回填）描述模糊的。
    *   **报价异常低廉**的。低价必然从产品（如用非阻氧管、劣质保温板）或工艺（减少施工步骤）上找补。

2.  **关键项目对比**：
    *   **锅炉**：对比**最小输出功率**和生活热水产率，这直接影响体验。
    *   **保温**：这是后期节能的核心。对比保温板品牌、厚度，反射膜是否为纯铝。
    *   **施工**：对比回填材料配比、打压测试承诺等。
    *   **价格**：将总价拆解为“锅炉设备费”和“每平米铺设费”进行比较，更清晰。

3.  **线上初步沟通感受**：
    *   客服或销售是否耐心、专业地解答你的疑问？
    *   是否主动邀请你看工地、看公司资质？
    *   沟通顺畅度如何？这将影响后续合作。

完成线上表格调研和初步沟通后，我们的目标不是在线上下定，而是**筛选出2-3家最靠谱的商家，进入线下考察阶段**。
