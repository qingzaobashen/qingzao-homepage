## 软装阶段
软装不是简单的买买买，而是一场关于如何具象化定义美好生活的旅程，把你对家的想象变成触手可及的日常。

### 1 全屋定制
1. 装修决策方法：
   - 步骤一：明确真实需求，分析家庭成员的生活习惯和收纳需求；
   - 步骤二：确定实现方案，选择适合的定制方式和材料；
   - 步骤三：评估方案缺点，考虑预算、工期、环保等因素的影响；
2. 全屋定制的两种主要方式：
   - 方式一：找专业的全屋定制商家，由商家负责设计、材料、加工和安装；
   - 方式二：请木工现场打柜子，自行购买材料，木工负责加工和安装；
3. 全屋定制的组成部分：设计+板材+加工（裁切、打孔、封边）+配件（五金、石材等）+安装；
4. 两种方式的对比：
   - 找商家定制：类似于硬装中的全包，省心但价格较高，是目前主流；
   - 木工现场打：类似于硬装中的清包，省钱但需要更多精力，且只能用EVA封边（用胶量大、易开裂）。
#### 1.1 全屋定制设计
1. 设计的重要性：
   - 确保定制家具符合房屋空间和物品尺寸；
   - 满足家庭成员的生活习惯和使用需求；
   - 实现美观的视觉效果，与整体装修风格协调；
2. 自助规划要点：
   - 分析家庭成员的生活习惯和收纳需求；
   - 考虑收纳方式，如抽屉、隔板、挂杆等的合理布局；
   - 预留嵌入式家电的位置和尺寸；
   - 确定外观风格，建议画草图或使用设计软件（如「酷家乐」app）辅助规划；
3. 设计师选择建议：
   - 考察设计师的沟通能力和理解需求的能力；
   - 查看设计师的实际案例，了解其设计风格和专业水平；
4. 注意事项：如果自助规划能够详细到可落地的程度，可以不请设计师，节省设计费用，但一般情况下自主设计无法准确把握配合尺寸与公差，还是建议请设计师。
#### 1.2 板材
1. **板材（饺子）=基材（馅儿）+饰面（皮儿）**
2. **基材**：木头➕胶。区别：木头粉末、颗粒、木条还是木板，以及什么品种的木头。基材挑选要注意胶的用量和品种。
3. **饰面**：区别：实木皮、纸皮、塑料皮贴面、刷漆、烤漆、刷木蜡油。
4. **板材性能**由基材和饰面的用料和工艺决定。
5. **板材选择原则**：板材没有绝对的好坏，只有是否适合自己的需求，需根据预算、使用场景、环保要求等因素综合考虑；
6. **选择板材的需求维度**：
   - 环保性：通常胶的用量越少越环保，关注甲醛释放等级（国标GB/T39600-2021）：E1级（≤0.124mg/立方米）、E0级（≤0.050mg/立方米）、ENF级（≤0.025mg/立方米）；
   - 稳定性：主要受基材结构影响，结构越紧密的板材稳定性越好，不易变形；
   - 握钉力：指板材与钉子的摩擦力，决定钉子是否牢固，受木材的强度、密度、硬度、弹性、纹理方向、含水率等因素影响，可通过观察木材结构判断；
   - 防潮性：受木材密度和饰面防水性能影响，潮湿环境（如厨房、卫生间）应选择防潮性能好的板材；
   - 美观度：与饰面图案、木材纹理颜色、表面加工工艺（如雕刻、封边）有关；
   - 价格：受材料成本、加工工艺等因素影响，一张标准板材的尺寸通常为1.22×2.44米；
   - 计算方式：全屋定制商家常用两种计算方式，展开面积（更准确）和投影面积（价格约为展开面积的四倍）；
7. 常见板材种类及特点：
   - 颗粒板（实木颗粒板或刨花板）：由木材颗粒与胶水压制而成；
     特点：结构稳定性较差，握钉力一般，防潮性能弱，环保等级适中，价格便宜；
     适用：预算有限的家庭，可用于非潮湿区域的柜体；
   - 欧松板（定向结构刨花板或OSB板）：由大刨片与MDI胶定向压制而成；
     特点：环保性能优于颗粒板，握钉力强，结构稳定；
     适用：墙面木饰面打底，也可用于柜体制作；
     价格：比颗粒板略贵；
   - 密度板（中纤板）：由木头粉末（木纤维）与大量胶水压制而成；
     特点：便于加工雕刻，可做复杂花纹，结构稳定性差，防潮性能弱；
     适用：常用于门板或背板，不建议用于潮湿区域；
   - 多层板（实木多层板或胶合板）：由多层木头薄片与胶水交错压制而成，家装常用9-11层；
     特点：用胶量相对较少，物理性能好，防潮性能强；
     适用：适合做柜体，尤其是厨房、卫生间等潮湿区域；
     价格：比颗粒板略贵；
     注意：若用胶量过大容易形变，一般不用于做柜门；
   - 生态板（大芯板或细木工板）：将软木木方粘贴成整板，上下贴一层薄木板；
     特点：易加工，但物理性能较差，防潮性能一般；
     适用：可用于简单的柜体制作，也是木工现场打柜子所用板材；
   - 实木板：由多块木板拼接而成（并非一整块木板），一般采用开放漆、水性漆或木蜡油处理；
     分类：横向拼接的为直拼板（质量更好），纵横交错拼接的为指接板；
     特点：环保性能好，质感优良，但价格较高，易受环境影响变形；
     适用：高端定制家具，适合对环保和质感要求高的家庭；
8. 板材相关名词解释：
   - 实木：指真实的木材，实木板材是由多块木材拼接而成；
   - 原木：指完整的、未经拼接的木头，也称为实木锯材；
   - 指接板和直拼板：都属于实木板，是实木板材的两种拼接方式；
   - 标准变化：国标GB/T28202-2011已废除，目前执行的是GB/T28202-2020标准；在GB/T28202-2020中，已不再使用"原木"这一说法，而是将原木、实木锯材、实木板统称为实木家具；
9. 板材宣传套路：
   - 市场上常见的"xxx实木板"、"实木xx板"（如实木颗粒板、多层实木板、生态实木板）等名称，实际上只是商家为了好听而使用的营销术语，并无实际的实木含量意义；
   - 严格意义上的实木板：仅指接板和直拼板；
   - 原木板：指未经拼接的完整木板，在市场上几乎买不到，可忽略；
10. 家具领域板材分类：
   - 实木类材料：包括原木和实木板；
   - 实木家具：根据新国标，无论是否使用原木，只要主要零部件由实木类材料制作的家具都称为实木家具（辅材可使用少量人造板材）；
   - 纯实木家具：市场上的常见说法，指全部由实木制作的家具，但国标中并无此定义；
   - 全实木家具（国标定义）：表面未经贴面处理的纯实木家具；
   - 覆面实木家具（国标定义）：表面经贴面处理的纯实木家具；
   - 人造板家具：由人造板材（如颗粒板、欧松板、密度板、多层板、生态板）制作的家具；
11. 板材类型选择：颗粒板和多层板优选。
方案一（不推荐）：
全部颗粒板。防潮性差。
方案二（推荐）：
厨卫柜体多层板，其他用颗粒板。
方案三（推荐）：
所有柜体全部多层板，所有柜门全部颗粒板。
12. 板材品牌选择：
进口：
爱格（Egger，奥地利），可丽芙（Cleaf，意大利），克诺斯邦（Kronospan，德国）等。
国产：
兔宝宝、千年舟、莫干山、露水河、大亚、大王椰、万华等。
本地品牌：
时间长有一定知名度。
13. 板材厚度：
通常用18厘（毫米），若多留空间背板选择9厘（毫米），注意单位是毫米。
14. 饰面
未做饰面的基材叫素板，做完饰面才可用到柜子上叫精板。
方式：贴面和涂料。
贴面：基材贴一层（三聚氰胺纸、PVC覆膜、PET吸塑和实木贴皮）上述的五种一般都贴面，最多是三聚氰胺纸，又叫免漆板。贴什么都OK，看个人喜好。
涂料：在基材做烤漆、刷漆或木蜡油。
烤漆和汽车上类似，但是贵。
15. 刷漆：
清漆和混油。前者是透明油漆，后者是带颜色油漆。要保留木材原本的色泽和纹理刷清漆，否则选混油。
开放漆和封闭漆。前者是将木材原本纹理色泽露出来，后者则是遮盖木材原本纹理色泽。P.S. 需要与清漆和混油区分开。清漆和混油是油漆，开放漆和封闭漆是工艺。
水性漆和油性漆。前者用清水稀释、更环保、主流选择，后者要用香蕉水稀释。
油漆是合成树脂为原料，晒干形成一层膜。木蜡油以植物油植物蜡为原料，渗透至木材，更环保美观。
#### 1.3 加工
1. 裁切、打孔和封边
2. 封边（重点）：把侧面裸露基材封起来。
方式：
- EVA热熔：用胶量大、不环保、胶线明显、易开裂。
- **PUR热熔：与EVA相反，正在取代EVA。这个挺好，性价比高。**
- 激光：最好，价格贵。
手工现场打柜子的精度没有机器高，一般用的也是EVA热熔封边。
#### 1.4 配件
   配件包括五金和台面
1. 五金：铰链（合页）、衣通、拉手、滑轨、拉直器、拉栏、碗篮以及橱柜和浴室柜的水槽水龙头。影响使用，一定要问清楚价格。
品牌：
国产：东泰（推荐）、悍高、顶固、诺米等
进口：百隆（Blum，奥地利）、格拉斯（Grass）、海蒂诗（Hettich，德国）、海福乐（Hafele）、萨郦奇（Salice，意大利）等
2. 台面：
只推荐石英石。不推荐不锈钢。岩板贵（冠珠 、一诺、中盛、德利丰、蒙娜丽莎，每平米2k-3k）。
3. 洗菜盆：304不锈钢台下盆
4. 洗手盆：陶瓷一体或可丽耐一体或岩板
#### 1.5 安装
1. 盯一下板材和配件是不是你选的，有没有色差，尺寸是否合适，安装完是否牢固顺畅就行了；
2. 安装过程中注意墙面和地面的保护，避免磕碰；
3. 安装完成后检查所有柜门和抽屉的开合是否顺畅，铰链是否牢固；
4. 检查五金配件是否齐全，有无损坏；
5. 清理现场垃圾，确保安装环境整洁。
#### 1.6 找定制
1. 想省心不想省钱的，直接索菲亚、欧派，
2. 想省钱的，让工长通过自己的渠道给你做，板材配件要自己把控；
3. 找全屋定制的代工厂，通过地图搜索，找到规模和口碑都不错的代工厂；搜一下”如何找到全屋定制的代工厂“；
4. 找到工厂实地看：1）看数控开料机开出的每张板是不是光滑没有崩边；2）看封边，白色板的封边，够不够光滑圆润，看不到胶线；3）看工厂有没有数控六面钻；4）看有没有环保证书；
5. 签订合同时注意明确板材品牌、环保等级、五金品牌、交货期和售后服务等条款。
### 2 家具
#### 2.1 成品家具
1. 成品家具的工艺和环保通常比定制家具要好；
2. 降低定制家具的比例，买成品家具更省钱； 
#### 2.2 木材
1. 成品家具也能让普通人用上实木家具；
2. 硬木与软木；
3. 硬木：北美黑胡桃，北美硬木之王；

#### 2.3 成品家具的选购建议
1. 能上实木上实木；能上硬木上硬木；
2. 网购推荐：源氏木语、维莎原木、原始原素、铜师傅等品牌；
3. 价格由高到低：
黑胡桃、樱桃木、橡木（非橡胶木，橡胶木是胶树）【源自橡树】、白蜡木、水曲柳、榉木（此处指山毛榉）、橡胶木、松木（多种木材统称，是软木）、杉木；
4. 选购时注意检查家具的结构是否稳固，抽屉和柜门的开合是否顺畅；
5. 查看家具的环保认证，如E0级、日本F4星级等；
6. 注意家具的尺寸是否符合房间空间，避免购买过大或过小的家具。
#### 2.4 为什么要用进口木材
1. 中国现在的森林覆盖率是23%，新中国成立时是8%，远低于世界平均水平31%；中国法令禁止砍伐天然林，适度发展速成林；
2. 进口木材多为天然林，生长周期长，材质更加稳定，不易变形开裂；
3. 进口木材的加工工艺和环保标准通常更高，品质更有保障；
4. 进口木材的种类更加丰富，可以满足不同风格和需求的家具制作；
5. 合理使用进口木材有助于保护国内森林资源，实现可持续发展。

### 3 织物
#### 3.1 窗帘
1. 安装方式：建议选择滑轨+窗帘盒的组合，美观且轨道隐藏，可配合电动滑轨实现智能控制；
2. 安装时机：滑轨需在硬装阶段安装，窗帘盒需在吊顶时预留；
3. 面料选择：
   - 涤纶：耐用、遮光、抗皱、易清洁、不透风、价廉，适用于卧室需要遮光静音的空间；
   - 亚麻：透光、通风、使用寿命长、清洗麻烦、易缩水，适用于无需遮光灰尘少的书房；
   - 棉麻：与亚麻类似，比亚麻柔软，透光、通风、不耐晒、不易清洗、不抗皱、易变形、性价比高，适合客厅；
   - 真丝：高级感强、具有一定阻燃性、价格昂贵，适用于高端装修的卧室或客厅；
   - 丝绒：垂感好、厚重、光泽亮丽、遮光、吸音、隔热保温、价格实惠、易沾灰，适用于卧室或客厅；
   - 纱帘：化纤原料制成的薄布，通风、透光柔和，适用于无需遮光的空间或作为布帘的辅助；
4. 尺寸选择：窗帘的高度低于2.8米时选择定高布，高于2.8米时选择定宽布；
5. 褶皱倍数：
   - 纱窗：建议1:2.5倍褶皱，保证通风和美观；
   - 布帘：宽度超过4米选2倍褶皱，小于4米选1.8倍褶皱，既美观又节约布料；
6. 遮光处理：如果需要更好的遮光效果，可以选择带有遮光衬的窗帘或在窗帘背面添加遮光布。
#### 3.2 沙发
1. 沙发别买贵的，性价比太低，建议将预算控制在合理范围内；
2. 选购要点：
   - 面料：优先选择好打理的面料，如科技布、防污布等，避免易脏难清洁的面料；
   - 坐深：建议选择坐深在50-60cm的沙发，既适合正坐也适合躺卧；
   - 回弹：选择回弹好的沙发，久坐不易塌陷，可通过按压沙发座垫测试；
   - 风格：与整体装修风格协调，避免风格冲突；
3. 购买渠道：推荐网购，价格相对透明，选择口碑好的品牌如顾家家居、左右沙发、芝华仕等；
4. 尺寸注意：根据客厅空间选择合适尺寸的沙发，确保沙发与茶几、电视柜等家具之间有足够的活动空间；
5. 结构检查：检查沙发框架是否稳固，可通过摇晃沙发测试，避免购买结构松散的沙发。
#### 3.3 床垫
1. 舒适度建议：选择硬中带软、软中带硬的床垫，既支撑性好又舒适，适合大多数人；
2. 床垫结构：主要由弹簧层、填充层、面料层三部分组成；
3. 弹簧类型：
   - 独立袋装弹簧：每个弹簧独立封装，翻身不影响旁边人，抗干扰性好，但透气性相对较差，部分承重区域可能老化较快；
   - 整网弹簧：所有弹簧连接成整体，支撑性好，透气性强，但抗干扰性差，翻身可能影响旁边人；
   - 分区弹簧：三分区设计已足够满足需求，多分区（如五分区、七分区）更多是营销噱头，实际效果差异不大；
4. 铺垫层材质：
   - 海绵：成本低，柔软度好，但长期使用易塌陷；
   - 乳胶：弹性好，支撑性强，透气性好，抗菌防螨，但价格较高；
   - 记忆棉：贴合身体曲线，缓解压力，但透气性差，夏天可能较闷热；
   - 注意：市场上很多床垫宣传的"乳胶床垫"可能只是添加了少量乳胶，选购时需注意乳胶层的厚度；
5. 面料层：选择透气、亲肤、抗菌防螨的面料，如针织面料、天丝面料等；
6. 选购建议：
   - 试躺：购买前一定要试躺，感受床垫的舒适度和支撑性；
   - 品牌推荐：栖作、雅兰、慕思、喜临门、金可儿等；
   - 环保：查看床垫的环保认证，如中国环保产品认证、欧盟CE认证等；
7. 更换周期：床垫建议每8-10年更换一次，使用时间过长会影响睡眠质量和健康。
### 4 家电
#### 4.1 吸油烟机
1. 风量就是吸烟量，20m³/min是足够的；风压是排风能力，300~400Pa；
2. 侧吸、顶吸之类的，离锅越近效果越好；
3. 选有自清洁功能、单向排烟、操作便捷的；
#### 4.2 燃气灶
1. 选火力大的；有效火力=热负载×热效率；热负载5kW就很大了，热效率1级能效66%
2. 集成灶相对难维修、难清洗；
#### 4.3 洗碗机
1. 类型：选嵌入式
2. 洗涤方式：区别不大，主流喷淋（推荐），高端超声波。
3. 烘干方式：区别不大，冷凝烘干就够，晶蕾烘干适合预算够
4. 容量：12套足够；越大越好。
#### 4.4 冰箱
1. 尺寸提前定好；定频变频：选变频。
2. 制冷方式：风冷和风直混冷；
3. 保鲜：低温、高湿度、低含氧量下；有应用保鲜技术就行
4. 循环系统：风机配蒸发器。单循环整个冰箱一套循环，双循环是冷藏和冷冻分开，三循环是冷藏和冷冻和变温分开。选后两者。
#### 4.5 洗衣机
1. 洗净原理：
   - 滚筒洗衣机：模拟手搓原理，洗净比高，磨损率低，省水但费电，适合洗涤高端衣物；
   - 波轮洗衣机：通过水流旋转带动衣物摩擦洗净，洗净比相对较低，磨损率高，费水但省电，适合洗涤普通衣物；
   - 选择建议：优先选择滚筒洗衣机，使用体验更好；
2. 电机选择：建议选择BLDC无刷直流电机，噪音小，寿命长，性能稳定；
3. 功能选择：
   - 杀菌功能：推荐选择带杀菌功能的洗衣机，如高温杀菌、银离子杀菌、UV杀菌等；
   - 烘干功能：如果预算充足，可选择洗烘一体机，节省空间；
   - 智能功能：如Wi-Fi控制、自动投放洗涤剂等，提升使用便利性；
4. 容量选择：根据家庭人口选择，3-4人家庭建议选择8-10kg容量，4人以上家庭建议选择10kg以上容量；
5. 品牌推荐：海尔、小天鹅、西门子、美的、松下等；
6. 注意事项：购买前测量预留空间的尺寸，确保洗衣机能够正常安装。
#### 4.6 烘干机
1. 方式：冷凝式、直排式和热泵式；只有热泵式才能即用即穿；
#### 4.7 热水器
1. 升数、水箱材质、是否恒温、零冷水、排烟方式；
2. 升数正常16升；无氧铜；水气双调恒温；零冷水可保恒温，可选；墙排式和平衡式；
3. 燃具8年报废；
#### 4.8 电视
1. 画质：电视>激光电视>投影；因为激光和投影都是把光投上去；
2. 尺寸选择：根据观看距离选择，观看距离 (cm)÷2.5=推荐电视尺寸 (英寸)；
3. 分辨率：选择4K分辨率，8K目前内容较少；
4. 面板类型：OLED画质好但价格高，LCD性价比高，推荐选择QLED或Mini LED的LCD电视；
#### 4.9 智能锁
1. 锁体类型：
   - 机械锁体：传统锁体，安全性高；
   - 电子锁体：智能控制，反应速度快；
2. 开锁方式：
   - 指纹识别：主流选择，建议选择半导体指纹识别，识别率高；
   - 密码解锁：方便访客使用，建议选择虚位密码功能；
   - 卡片解锁：适合老人和小孩；
   - 钥匙解锁：必备备用方式；
   - 人脸识别：高端功能，识别速度快，但价格较高；
3. 选择要点：
   - 安全性：选择C级锁芯，具备防撬、防暴力开启功能；
   - 稳定性：选择知名品牌，避免频繁故障；
   - 续航能力：电池续航至少6个月以上；
   - 防小黑盒：具备防电磁干扰功能；
4. 品牌推荐：凯迪仕、德施曼、鹿客、小米、飞利浦等；
5. 安装注意：购买前确认门的厚度和类型是否适合安装智能锁；
#### 4.10 空调
1. 类型选择：
   - 分体式空调：适合单个房间使用，安装灵活，价格实惠；
   - 中央空调：适合多房间使用，美观大方，但安装成本高；
   - 风管机：介于分体式和中央空调之间，适合小户型；
2. 匹数选择：根据房间面积选择，1匹适合10-15㎡，1.5匹适合15-20㎡，2匹适合20-30㎡；
3. 能效等级：选择一级能效，节能省电；
4. 功能选择：
   - 变频技术：舒适省电，建议选择变频空调；
   - 除湿功能：适合潮湿地区；
   - 自清洁功能：减少清洗麻烦；
   - 静音设计：适合卧室使用；
5. 品牌推荐：格力、美的、海尔、海信、大金等；
6. 安装注意：选择正规安装公司，确保安装质量和售后服务；
#### 4.11 家电选购总结
1. 不同品牌：冰箱洗衣机--海尔；分体式空调--格力；洗碗机--美的；烟机灶具--方太；集成灶--火星人；智能锁--凯迪仕；
2. 线上线下都了解一下，团购也行；
3. 购买时机：选择节假日促销，如618、双11等，可获得更多优惠；
4. 售后服务：选择售后服务完善的品牌，确保使用无忧；
