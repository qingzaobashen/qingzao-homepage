# 扫地机器人挑选指南

### 第一部分：清洁能力

清洁能力是扫地机器人的根本。

| 选购维度 | 要 √ | 不要 × | 关键解析与建议 |
| :--- | :--- | :--- | :--- |
| **1. 防缠绕能力** | **主刷设计有防缠绕功能**（如胶毛一体滚刷、纯胶刷或特殊的防缠绕结构）。 | **无防缠绕设计的传统毛刷**。 | **这是首要痛点！** 对于长发、宠物家庭，滚刷缠绕是最大困扰。优选具有主动切割或特殊结构设计的滚刷。 |
| **2. 吸力大小** | **吸力 ≥9000Pa**，**当前主流产品多在10000Pa以上**。 | **吸力 <9000Pa** 的过时或低端型号。 | 吸力决定了对灰尘、颗粒物的回收效率。吸力越大，清洁能力越强，尤其是地毯清洁。 |
| **3. 边角清洁** | **带“机械臂”或“动态边刷”** 等延伸清洁技术。 | 只有基础扭动功能的边刷。 | 传统圆形机身无法完全贴紧墙边，形成清洁死角。**机械臂技术**能让边刷或拖布伸出，实现真正的贴边清洁，是质变功能。 |
| **4. 集尘方式** | **双边刷**。 | **单边刷**。 | 双边刷能更高效地将垃圾聚拢到主吸口，清扫路径覆盖更全面，效率更高。 |

### 第二部分：智能导航和避障

再强的吸力，如果经常“犯傻”、被困或漏扫，也毫无意义。

| 选购维度 | 要 √ | 不要 × | 关键解析与建议 |
| :--- | :--- | :--- | :--- |
| **5. 避障技术** | **双目视觉避障** 或 **结构光+AI视觉融合避障**。 | **机械碰撞避障**（已基本淘汰）或 **红外避障**（精度低）。 | **避障能力决定体验下限**。优秀的避障能精准识别拖鞋、袜子、电线、宠物粪便等，避免推着走、缠绕或“泼粪”悲剧。**双目视觉是当前高端主流**。 |
| **6. 导航与建图** | **LDS激光雷达导航**（主流成熟）或 **3D TOF雷达导航**（精度更高）。 | **视觉导航**（对光线依赖强，稳定性差）或无规律随机碰撞。 | 激光导航建图快、精度高。高端TOF方案在暗光环境下表现更佳。精准的地图和路径规划是高效全屋覆盖的基础。 |
| **7. 越障能力** | **越障高度 ≥20mm**。 | 越障能力弱（<15mm）。 | 确保机器人能轻松翻越推拉门轨道、卫生间门槛等家中常见低矮障碍。 |

### 第三部分：基站功能

基站功能决定了你需要为它“服务”的频率，是实现高度自动化的核心。

| 选购维度 | 要 √ | 不要 × | 关键解析与建议 |
| :--- | :--- | :--- | :--- |
| **8. 自动集尘** | **基站带自动集尘功能**，尘袋容量大（建议>2L）。 | **无自动集尘功能**。 | **核心解放双手功能**。每次清洁后，机器人自动将尘盒垃圾抽到基站的大尘袋中，你只需**每月甚至数月更换一次尘袋**，彻底告别频繁清理脏尘盒的烦恼。 |
| **9. 自动洗拖布** | **基站带热水洗、搓、烘干拖布功能**。 | 手动清洗拖布或仅简单湿润拖布。 | 自动清洗并烘干拖布，避免异味和细菌滋生。**热水洗和搓洗功能**去污能力远胜冷水冲洗。 |
| **10. 自动上下水** | **基站支持自动上下水模块安装**（需预留位置）。 | 仅支持手动水箱换水。 | **终极解放双手功能**。如同洗衣机，自动加清水、排污水，实现长达数月的完全无需人工干预的清洁，特别适合大户型。 |
| **11. 电池容量** | **≥5200mAh**（适用于60㎡以上户型）。 | 容量过小，中途需回充。 | 大容量电池确保一次性完成全屋清洁，提高效率。 |

### 第四部分：品牌选择

不同品牌在技术路线和产品特点上差异显著。

| 品牌 | 优势/特点 √ | 缺点/痛点 × | 选购建议 |
| :--- | :--- | :--- | :--- |
| **石头 (Roborock)** | **防缠绕技术突出**，算法稳定，导航路径规划优秀。 | 曾有机型出现“**容易进水，刮地板，集尘声音大，风机老异常**”等问题（需关注新型号是否改进）。 | **推荐**：注重**毛发防缠绕**、算法稳定性的用户首选。重点关注其最新旗舰或主打型号。 |
| **科沃斯 (Ecovacs)** | 市场占有率高，产品线丰富，部分型号AI避障技术领先。 | 部分型号存在“**画图不行，智障，没法自己回基站，水痕，维修费高**”等负面反馈，品控和算法稳定性口碑分化。 | **建议**：可关注其高端旗舰型号（如X2系列），但需仔细查看真实用户的长测评价，中低端型号需谨慎。 |
| **追觅 (Dreame)** | 参数“卷王”，**功能配置齐全**（吸力、热水洗、机械臂等），性价比高。 | 可能存在“**功能齐全，但样样不精**”的情况，整体调教、算法细节和长期稳定性有待市场进一步检验。 | **推荐**：追求**高参数、新功能**且预算有限的“技术尝鲜者”。 |
| **云鲸 (Narwal)** | 早期以“自动洗拖布”开创品类，拖地体验曾受好评。 | **“吸力不行，清理尘盒频繁”**，且存在“**商场沿专柜，品控差**”的评价。 | **谨慎选择**：在竞品全面发展的今天，其核心优势不再明显，且曝出短板较多，建议优先考虑其他品牌。 |

### 选购清单

**第一步：明确需求**
*   家里毛发多吗？优先选石头。
*   讨厌倒垃圾？选带自动集尘的基站。
*   想完全不管？自动上下水是最终选择。
*   户型大小决定电池和续航需求。

**第二步：看关键参数**
1.  **基础配置**：吸力10000Pa以上、激光/TOF导航、双目或AI视觉避障。
2.  **解决痛点**：防缠绕主刷、机械臂贴边清洁。
3.  **省心功能**：自动集尘、热水洗烘拖布。

**第三步：选品牌**
*   **稳定、毛发清理好**：选石头的旗舰或次旗舰。
*   **避障好、想尝鲜**：看看科沃斯最新旗舰的用户反馈。
*   **性价比高、功能新**：选追觅的顶配型号。

**最后建议**：参数好看不一定好用，线上选好1-2款后，一定要看真实用户用了3个月以上的评测，重点关注：
*   用久了避障还准吗？
*   滚刷和边刷容易缠毛发吗？
*   基站容易发臭吗？好清洁吗？
*   APP用起来顺吗？稳定吗？
